# Cloudelligent - Frontend Developer Technical Assessment

Angular application for managing Products with CRUD operations.

# Implemented Features

# Required
Part 1 - Practical Task: Full CRUD for Products.
Part 2 - Caching: localStorage persistance(data is saved and loaded across page reloads)
Part 3 - Optimization
    - Minimal bundle size (single standalone component, no external libraries)
    - Efficient rendering for small dataset

# Bonus
    - Form validation with real time error messages
    - Dark/Light theme toggle

# How to Run
```bash
npm install
ng serve
